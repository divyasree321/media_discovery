# 🎬 Media Discovery Platform

A modern and responsive Media Discovery Platform built using pure HTML and CSS. This project showcases trending movies, popular TV shows, and a contact section while demonstrating responsive web design principles.

## 🚀 Live Demo

https://divyasree321.github.io/media_discovery/

## 🌐 Deployment

This project is deployed using GitHub Pages.

### Steps:

1. Push files to GitHub repository.
2. Open Repository Settings.
3. Navigate to Pages.
4. Select:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
5. Save changes.

GitHub will generate a live website URL automatically.
